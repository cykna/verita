O que é o Verita?

O Verita é um app de comunicação, mas pensado do zero para ser modular, flexível e escalável.
Nada de monólito inchado. Aqui, cada recurso é um módulo — entra quem agrega, sai quem atrapalha.

A ideia é simples: controle total da experiência, tanto para usuários quanto para desenvolvedores. A ideia principal
se baseia em você poder hospedar a sua própria versão do verita, assim como feito antigamente.
A ideia do verita é ser somente a parte do servidor que lida com recebimentos do client, mas questões como UI e banco de dados,
você é quem escolhe.
Isso é feito por meio de um protocolo de mensagens definido em specs/protocol.md. Nele é dito o que o Server espera e o que o Storage espera.
Seguindo isso, qualquer um pode fazer envio/recebimento pra qualquer host.
