O que é o Verita?

O Verita é um app de comunicação, mas pensado do zero para ser modular, flexível e escalável.
Nada de monólito inchado. Aqui, cada recurso é um módulo — entra quem agrega, sai quem atrapalha.

A ideia é simples: controle total da experiência, tanto para usuários quanto para desenvolvedores.
